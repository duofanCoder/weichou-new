# 项目使用

backend 目录是后端，使用idea 打开，依赖安装好后，run就可以，数据库创建好，不需要创建表，会自动生成

frontend 目录是前端，使用vite2，编译打包迅速，使用 yarn 命令或者npm i 命令安装依赖，然后使用yarn dev  或者 npm run dev 运行前端即可，
